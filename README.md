# Markuse asjade integratsioonirakendus

Rakendus Android operatsioonsüsteemi jaoks, mis võimaldab arvutit kaughallata. Kasutab RESTful päringud [MasAPI](https://github.com/MarkusMaal/mas-tui/tree/main/MasAPI) vastu.

Projekt on Alpha arendusjärgus.

Implementeeritud funktsionaalsus:

- Markuse arvuti asjade väljaande info kuvamine
- Konfiguratsiooni kuvamine
- Dünaamiline rakenduse ikoon
